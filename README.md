# Aplikasi Less Paper

Aplikasi verrifikasi berkas  secara less paper pada proses pengurusan kenaikan pangkat , pensiun , karpeg dan karis/karsu

Feature:
1. Search arsip by NIP
2. Display data utama
3. Verifikasi berkas

Requirement:
1. PHP 5
2. Mysql 5
3. Database Mirror
4. Database DMS
5. Database Takah
